# mask 服务

# 1. 简述
mask 服务的核心功能就是提供一个静态文件服务，客户端上传一个任意格式文件，服务对文件进行处理并返回文件位置
> * 静态图片服务
> * 音视频相关服务

# 2. API
### 上传文件
```
|    参数    |       说明          |           类型             |      默认值          |
+-----------+---------------------+---------------------------+---------------------+
|   type    |指定上传文件，文件地址  |         string             |       all           |
|   data    |   文件              |         FileList           |       ---           |
|   ts      |     时间戳          |         timestamp          |       ---           |
+-----------------------------------------------------------------------------------+
|    返回    |                    |                            |                     |
+-----------+---------------------+---------------------------+---------------------+
|   code    |      状态           |        number              |      1 ok           |
|   path    |      文件地址        |         string[]           |     -------         |
```
